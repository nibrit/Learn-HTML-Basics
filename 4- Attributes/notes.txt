# Attributes:
-> The additional information about the tag is called as attributes.
OR
-> The properties of a tag is called as attributes.
-> We have to always provide the attributes inside the opening tag of HTML element.

# Types of Attributes:
-> Following are some main types of attributes:
    1. Core Attributes
    2. Generic Attributes
    3. Global Attributes

1. Core Attributes:
-> Core attributes are the fundamental attributes.
-> Means most of the time we are using these attributes.
-> For Ex: id, class, style, title
   1. id = To provide unique identification (#id)
   2. class = To access or target the multiple elements at a time. (.className)
   3. style = To provide the inline css.
   4. title = It is used for tooltip. (Internal Information)

2. Generic Attributes:
-> Generic attributes are used only with special tags.
-> For Ex: src, alt, href, width & height
    1. src & alt = Image tag
    2. href = Anchor Tag

3. Global Attributes:
-> Global attributes we can use with most of the elements.
-> For Ex: id, class, style, title, tabindex, accesskey, contenteditable, spellcheck, dragable, translate, lang, etc.
