# Introduction to HTML:

-> HTML Stands for Hyper Text Markup Language.
-> Hyper = Reference / Link
-> Text = Data / Information
-> Markup = Displaying on the browser / UI in the form of Predefined Tags.
-> Language = Communication purpose
-> HTML is used to create or develop the structure of a webpage.
-> We have to use .htm or .html extension to create html webpages.
-> We can create html webpages by using simple editors also. (VS Code)

# Structure of HTML:

-> The structure of HTML is divided mainly into 2 parts: 1. head 2. body

1. head: The head contains the information about the html document.
   For Ex: meta tags, title, links, etc.

2. body: The body contains the data which you want to display on the browser.

# Meta Tags:

-> Meta tags are used to provide the information about HTML document to the browser.
-> Metadata: The data about data is called as Meta data.
-> Meta tags are defined inside the <head> tag.

# HTML Tags:

-> In HTML everything will be considered as Tags.
-> The predefined keywords which is enclosed in between '<>' is called as Tag in HTML.

-> There are mainly two types of tags:

1. Paired Tags / Container Tags:
2. Un-paired Tags / Non-container Tags / Self-closing Tags:

3. Paired Tags / Container Tags:
   -> The tag which has opening tag along with its closing tag is called paired tags.
   -> It is also known as container tags.
   -> For Ex: <html>...</html>, <head>...</head>, All Headings tags, paragraph tag, etc.

4. Un-paired Tags / Non-Container Tags / Self-Closing Tags:
   -> The tag which don't have its separate closing tag is called as un-paired tags.
   -> It is also known as non-container tags.
   -> It is also known as self-closing tags.

Self-Closing Tag: The tag which is capable to close itself is known as self-closing tags in HTML.
-> For Ex: <meta/>, <br>, <hr>, <img/>, <input/>, etc.

# Headings Tags:

-> Headings tags are used to provide the headings in HTML document.
-> We have can provide the headings in 6 ways:
-> <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
-> When we go from <h1> to <h6> the size of the headings tags will decrease.
-> Largest Heading Tag -> <h1>
-> Smallest Heading Tag -> <h6>

# Paragraph Tag:

-> The paragraph tags are used to provide the paragraph to the HTML document.
-> We can provide the multiple paragraph tags in a document.
